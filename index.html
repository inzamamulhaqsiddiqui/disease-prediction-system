Welcome to the Disease Prediction System

This system helps users predict the most likely disease based on symptoms using Machine Learning and a trained prediction model. Simply select your symptoms from the list and our intelligent system will analyze the inputs and provide the predicted disease instantly.

Key Features

  • ✔ Easy-to-use interface for selecting symptoms
  • ✔ Accurate disease prediction using trained ML model
  • ✔ Stores user prediction history for future reference
  • ✔ Fast and secure Django-based application

How to Use

  1. Navigate to the Disease Prediction page from the menu.
  2. Select symptoms such as fever, headache, nausea, etc.
  3. Click the Predict button.
  4. View the predicted disease and check your medical history anytime.

This platform is developed to make early disease detection simpler and faster, empowering users to take timely medical decisions.
